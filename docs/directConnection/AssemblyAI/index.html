AssemblyAI

AssemblyAI

Properties used to connect to AssemblyAI.

assemblyAI​

  • Type: true | audio?: true
  • Default: {audio: true}

Connect to Assembly AI's speech recognition API to transcribe your audio.

Example​

  • Sample code
  • Full code
<deep-chat
  directConnection='{
    "assemblyAI": {
      "key": "placeholder key",
      "audio": true
    }
  }'
></deep-chat>
