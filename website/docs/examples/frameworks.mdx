## Angular

Please add the `CUSTOM_ELEMENTS_SCHEMA` into your module's `schemas` array and import the component
as `import 'deep-chat';`. See this [blog](https://www.thisdot.co/blog/how-to-integrate-web-components-using-lit-in-angular) for more information.

[Click for Live Example](https://codesandbox.io/s/deep-chat-angular-mk2v62?file=/src/app/app.component.html)

## Troubleshooting

If your platform has not been listed and Vanilla JS approach does not work as fallback, or if you are having issues with any of
the existing examples; please see [**github issues**](https://github.com/OvidijusParsiunas/deep-chat/issues) or create
a [**new issue ticket**](https://github.com/OvidijusParsiunas/deep-chat/issues/new) and we will look into
it as soon as possible.
