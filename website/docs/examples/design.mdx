# Design

Examples to help you get started in styling your Deep Chat component.

### ChatGPT {#chatGPT}

This example simulates the styling used in [ChatGPT](https://chat.openai.com/). If you are using a custom avatar - please replace the _"path-to-icon.png"_ string to your file's path or
