# DeepSeek

Properties used to connect to [DeepSeek AI](https://www.deepseek.com/).

### `deepSeek` {#deepSeek}

- Type: \{<br />
  &nbsp;&nbsp;&nbsp;&nbsp; `model?: string`, <br />
  &nbsp;&nbsp;&nbsp;&nbsp; `temperature?: number`, <br />
  &nbsp;&nbsp;&nbsp;&nbsp; `max_tokens?: number`, <br />
  &nbsp;&nbsp;&nbsp;&nbsp; `top_p?: number`, <br />
  &nbsp;&nbsp;&nbsp;&nbsp; `frequency_penalty?: number`, <br />
  &nbsp;&nbsp;&nbsp;&nbsp; `presence_penalty?: number`, <br />
  &nbsp;&nbsp;&nbsp;&nbsp; `stop?: string | string[]`, <br />
  &nbsp;&nbsp;&nbsp;&nbsp; `system_prompt?: string` <br />
  \}
- Default: _\{model: "deepseek-chat", temperature: 1, max_tokens: 4096\}_

Connect to DeepSeek's [`chat completion`](https://api-docs.deepseek.com/api/create-chat-completion) API. <br />
`model` is the DeepSeek model to use ("deepseek-chat" or "deepseek-reasoner"). <br />
`temperature` controls randomness (0.0-2.0). Higher values produce more creative outputs. <br />
`max_tokens` limits the maximum number of tokens in the response (1-8192). <br />
`top_p` controls diversity through nucleus sampling (0.0-1.0). <br />
`frequency_penalty` penalizes tokens based on their frequency in the text (-2.0 to 2.0). <br />
`presence_penalty` penalizes tokens based on their presence in the text (-2.0 to 2.0). <br />
`stop` defines sequences where the API will stop generating (up to 16 sequences). <br />
`system_prompt` provides behavioral context and instructions to the model. <br />
