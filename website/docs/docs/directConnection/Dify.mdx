Properties used to connect to [Dify](https://dify.ai/).

### `dify` {#dify}

- Type: `true` | \{ `inputs?: object`, `user?: string`, `files?: {type: "file"|"collection", id: string}[]`\}

- Default: _\{user: "user"\}_

Connect to [Dify's Chat API](https://docs.dify.ai/en/guides/application-publishing/developing-with-apis). You can set this property to _true_ or configure it using an object: <br />
`inputs` is an object containing custom input variables defined in your Dify application. <br />
`user` is the user identifier for the conversation. <br />
`files` is an array of pre-uploaded file references to include in the conversation. Each object requires a `type` ("file" or "collection") and an `id` referencing the uploaded file. <br />

### Getting Your API Key

To use Dify with Deep Chat, you need to:

1. **Create a Dify account** at [dify.ai](https://dify.ai/) or set up a [self-hosted instance](https://docs.dify.ai/en/getting-started/install-self-hosted)
2. **Create an application** (Chatbot, Agent, or Workflow)
3. **Get your API key** from the application's API Access page

:::tip
Your API key can be found in your Dify application under **API Access** in the left sidebar. See the [Dify API documentation](https://docs.dify.ai/en/guides/application-publishing/developing-with-apis) for more details.
:::
