# Installation

Install the component via [npm](https://www.npmjs.com/):

```bash
npm install deep-chat
```

For React, install the following instead: <br />

```bash
npm install deep-chat-react
```

Access the component via CDN:

```
https://unpkg.com/deep-chat@2.4.1/dist/deepChat.bundle.js
```

When coding with AI, use [llms.txt](https://github.com/OvidijusParsiunas/deep-chat/blob/main/llms.txt) as a reference:

```
"Use https://github.com/OvidijusParsiunas/deep-chat/blob/main/llms.txt to add a chat component to my website."
```
